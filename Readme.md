### 推荐安装yarn
1. 确保 node 版本是 8.10 或以上
2. npm i yarn -g

### 快速开始


进入目录安装依赖:

```bash
yarn install
```

启动服务.

```bash
$ yarn start
```

编译.

```bash
$ umi build
```
